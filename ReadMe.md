# Frontend Resources for My Students

This guide is a curated starting point for students learning frontend development with me as part of a Java Full Stack path.

The goal is simple:

- build a strong foundation in HTML, CSS, JavaScript, Git, TypeScript, and React
- learn from reliable documentation
- practice with a few high-value resources instead of getting lost in too many links
